================= SCOPE OF THE PASSWORD AUTOMATION SCRIPT: =====================

This password automation script scans accounts that are part of an Active Directory.
If the script determines that a account either failed 3 login attempts or if the password is more than 90 days old.
The script AUTO LOCKS the account. IT will AUTO Generate and send a email support ticket for each violation
to the Helpdesk identifying which accounts are locked.
